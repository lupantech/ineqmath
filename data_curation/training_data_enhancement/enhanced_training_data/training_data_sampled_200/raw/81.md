# Problem Entry

## Problem Type
bound

## Problem
Let $\alpha_{i}>0, i=1,2, \ldots, n$, be real numbers such that $\alpha_{1}+\alpha_{2}+\cdots+\alpha_{n}=1$. Determine the maximal constant $C$ such that the following inequality holds for all $\alpha_{i}$:
$$
\alpha_{1}^{\alpha_{1}} \alpha_{2}^{\alpha_{2}} \cdots \alpha_{n}^{\alpha_{n}} \geq C.
$$

## Current Solution
If we take $a_{i}=\frac{1}{\alpha_{i}}, i=1,2, \ldots, n$, by the Weighted $AM-GM$ inequality we get
$$
\frac{1}{\alpha_{1}^{\alpha_{1}}} \frac{1}{\alpha_{2}^{\alpha_{2}}} \cdots \frac{1}{\alpha_{n}^{\alpha_{n}}} \leq \frac{1}{\alpha_{1}} \alpha_{1}+\frac{1}{\alpha_{2}} \alpha_{2}+\cdots+\frac{1}{\alpha_{n}} \alpha_{n}=n
$$
i.e.
$$
\frac{1}{n} \leq \alpha_{1}^{\alpha_{1}} \alpha_{2}^{\alpha_{2}} \cdots \alpha_{n}^{\alpha_{n}}
$$
Equality holds when $\alpha_1 = \alpha_2 = \cdots = \alpha_n = \frac{1}{n}$, in which case $\alpha_1^{\alpha_1} \cdots \alpha_n^{\alpha_n} = \left(\frac{1}{n}\right)^1 = \frac{1}{n}$. This gives the minimum value of the product, so $C = \frac{1}{n}$ is maximal.

Therefore, the answer is $C = \frac{1}{n}$.

## Expected Answer
$C = \frac{1}{n}$

## Enhanced Solution
Enhanced Solution:

We wish to find the largest constant C such that for every choice of positive real numbers \(\alpha_{1},\dots,\alpha_{n}\) with \(\alpha_{1}+\cdots+\alpha_{n}=1\), the inequality
\[
    \alpha_{1}^{\alpha_{1}}\,\alpha_{2}^{\alpha_{2}}\cdots\alpha_{n}^{\alpha_{n}}\ge C
\]
holds.

Step 1: Reformulate via reciprocals and prepare to apply the weighted AM–GM inequality.

Define for each \(i\)
\[
    x_i := \frac1{\alpha_i},
    \quad w_i := \alpha_i.
\]
Observe that
\[
    x_i>0,
    \qquad w_i>0,
    \qquad \sum_{i=1}^n w_i = \sum_{i=1}^n \alpha_i = 1.
\]
We will apply the weighted AM–GM inequality in the form:

  If \(w_i\ge0\) with \(\sum w_i=1\) and \(x_i>0\), then
\[
    \sum_{i=1}^n w_i x_i \;
    \ge\; \prod_{i=1}^n x_i^{w_i},
\]
with equality precisely when all \(x_i\) are equal.

Step 2: Apply weighted AM–GM.

Using the choices above,
\[
    \sum_{i=1}^n w_i x_i
    = \sum_{i=1}^n \alpha_i \cdot \frac1{\alpha_i}
    = \sum_{i=1}^n 1
    = n.
\]
Meanwhile,
\[
    \prod_{i=1}^n x_i^{w_i}
    = \prod_{i=1}^n \bigl(\tfrac1{\alpha_i}\bigr)^{\alpha_i}
    = \frac{1}{\prod_{i=1}^n \alpha_i^{\alpha_i}}.
\]
Therefore, the weighted AM–GM inequality gives
\[
    n \;\ge\; \frac{1}{\prod_{i=1}^n \alpha_i^{\alpha_i} }
    \quad\Longrightarrow\quad
    \prod_{i=1}^n \alpha_i^{\alpha_i} \;\ge\; \frac1n.
\]

Step 3: Check the equality case.

Equality in the weighted AM–GM step occurs if and only if
\(x_1 = x_2 = \cdots = x_n\), i.e.
\[
    \frac1{\alpha_1} = \frac1{\alpha_2} = \cdots = \frac1{\alpha_n}.
\]
Combined with the constraint \(\alpha_1+\cdots+\alpha_n=1\), this forces
\[
    \alpha_1=\alpha_2=\cdots=\alpha_n=\frac1n.
\]
Substituting back yields
\[
    \prod_{i=1}^n \alpha_i^{\alpha_i}
    = \Bigl(\frac1n\Bigr)^{\sum_{i=1}^n\alpha_i}
    = \Bigl(\frac1n\Bigr)^1
    = \frac1n.
\]
Thus the bound \(1/n\) is indeed attained, and no larger constant can satisfy the inequality for all admissible \(\alpha_i\).

Conclusion:

The maximal constant is \(C=1/n\).  
The answer is $C=\frac{1}{n}$
